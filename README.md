# Mortalidade_no_Brasil
Análise da mortalidade no Brasil : Envelhecimento populacional, DCNTs e desigualdades territoriais. Foco nos anos de 2000 e 2020.

# Metodologias e Insights
  Para realizar a análise da mortalidade no Brasil, avaliando principalmente os anos de 2000 e 2020, inicialmente extraí uma base de dados contendo o número de óbitos por ano segundo a causa básica (Capítulo CID-10), através do sistema Tabnet. A base foi importada para o Google Colab, assim como as bibliotecas python que foram utilizadas no processo de manipulação, tratamento, análise e visualização dos dados.  
	A primeira análise realizada visou comparar as principais causas de morte dos anos 2000 às de 2020, avaliando o processo de transição epidemiológica. Para a realização da comparação, a base foi manipulada para gerar dados em porcentagem, demonstrando quantos % cada causa representou do total de óbitos do ano. Com os novos dados, gerei um gráfico de barras agrupadas, que permitiu a visualização dessa transição e das causas com maior impacto nos óbitos de cada ano.  
	Ao identificar as causas mais importantes e as que sofreram aumento expressivo em 2020 comparadas a 2000, iniciei as análises mais aprofundadas baseadas nesses dados e em informações populacionais consolidadas relevantes, fornecidas pelo IBGE. O primeiro tópico dessas análises foi voltado ao envelhecimento populacional, no qual utilizei uma base de dados contendo os óbitos por faixa etária segundo a causa, também extraída do Tabnet. A partir da base, gerei um gráfico de linhas que demonstrou a prevalência das causas de acordo com a faixa etária, no qual foi possível notar nitidamente o perfil de uma população mais envelhecida, conforme o esperado.  
	O gráfico de óbitos por faixa etária segundo a causa também evidenciou o impacto das causas externas nos óbitos de 2020, demonstrando que esse ainda deve ser um ponto de atenção para as políticas públicas. A fim de detalhar esse ponto, extraí uma nova base de dados do Tabnet, contendo o número de óbitos por causas externas segundo a Unidade Federativa, no ano de 2020. Com ela, gerei um gráfico de barras que facilitou a visualização da variação desses números de acordo com a região do país e a identificação das UF com as maiores taxas de mortalidade por causas externas. Para obter dados mais precisos e mais eficazes no subsídio de ações em saúde pública, extraí uma base de dados contendo os números de óbito por cada uma das causas externas, nas UF de Roraima e Ceará (que apresentaram as maiores taxas de mortalidade no gráfico anterior), podendo assim identificar quais devem ser os focos das ações.  
	Com essas análises, foi possível ressaltar 3 pontos que podem orientar políticas que contribuam para a construção de um sistema de saúde pública mais efetivo: o envelhecimento populacional, a prevalência e impacto das DCNTs e a persistência de óbitos por causas externas, principalmente em determinadas regiões do país.
